## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

1. Install all dependencies using the `npm install` command.
2. Start the development server using the `node express_server.js` command.
3. Swear that this will work and not crach your machine.
